# development-images

本喵使用的開發環境打包的容器

# background

開發環境的搭建是一個麻煩的事情，此外在西朝鮮要爲開發環境解決網路封鎖同樣讓人沮喪，好在容器技術的發展爲這些問題提供了解決方案。

使用容器可以將開發環境和代理工具一起打包，要在新的地方恢復開發環境只需要運行容器即可，本喵已經這麼做了多年(始於2020年)，以實踐經驗來看這是一個行之有效的好主意，於是本喵決定把自己的經驗分享出來，同時把使用到的容器也一起分享在此
