[frp](https://github.com/fatedier/frp)，它是一個內網穿透工具，例如你家或公司沒有提供公共的訪問
ip，但你有一個公網服務器，可以使用它把私有的家裏/公司服務穿透到公網。

此外對於西朝鮮用戶也可以使用它來配合 v2ray 之類的軟體一起使用
